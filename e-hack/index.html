프로그램 소개

js코드 작성이 귀찮을 때 사용할 수 있는 프로그램입니다.

지금도 그렇고 앞으로도 그럴꺼지만 기초적인 기능만 지원하고 나머지는 js코드로 작성해야 합니다.
















사용법

1. 원하는이름.txt를 생성합니다.

2. 코드를 작성합니다.

3. cmd를 열어서 E-HACK <파일이름> 을 입력합니다.


-- 코드 작성법 --


1. 작 내에 변수 수정


1-1. 변수 이름으로 수정(비추천)

#변수이름#="변수 내용"


1-2. id로 수정(추천)

$변수이름$="변수 내용"


1-3. 변수 번호로 수정

%변수이름%="변수 내용"

============

1. 변수 선언 및 수정

@변수이름@="변수 내용"

============

2. if문

^변수 또는 값^:^조건(<,>,=)^:^변수 또는 값^={

...

}


작성예:

^@a@^:^=^:^"1"^={

$d1eg$="100"

}

============

3. js코드

_js코드_


작성예:

_var a={_

_ a:1,_

_ b:2_

_}_

@b@="3"

_b+=a.a+a.b;_

_console.log(b);_

============

4. 함수


4-1. 작성

!함수 이름!={

...

}


4-2. 실행

!함수 이름!
